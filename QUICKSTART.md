# Quick Start Guide

## Getting Started in 3 Simple Steps

### Step 1: Install Dependencies
```bash
npm install
```

### Step 2: Start Development Server
```bash
npm start
```

### Step 3: Open in Browser
The site will automatically open at `http://localhost:3000`

## What's Next?

### Customize Your Portfolio

1. **Update Personal Info**
   - Open `index.html`
   - Update name, email, location
   - Add your social media links

2. **Add Your Photo**
   - Place your photo in `assets/img/`
   - Update the profile placeholder in `index.html`

3. **Add Your CV**
   - CV already placed in `assets/pdf/CV_Tnp.pdf`
   - Download button will work automatically

4. **Customize Colors**
   - Open `assets/css/style.css`
   - Modify CSS variables in `:root` section

5. **Add Project Images**
   - Place project screenshots in `assets/img/`
   - Update project cards in `index.html`

## Deploy Your Site

### Option 1: Vercel (Recommended)
```bash
npm install -g vercel
vercel
```

### Option 2: GitHub Pages
1. Create a GitHub repo
2. Push your code
3. Enable GitHub Pages in repo settings

### Option 3: Netlify
1. Visit netlify.com
2. Drag and drop your project folder
