# Image Assets

Place your images here:

- Profile photo
- Project screenshots
- Background images
- Icons and logos

## Recommended Formats

- Profile photo: JPG/PNG (500x500px or higher, square aspect ratio)
- Project images: JPG/PNG (1200x800px or higher)
- Logos: PNG with transparency (256x256px or higher)

## Optimization

Please optimize images before adding them to reduce file size:
- Use tools like TinyPNG, ImageOptim, or similar
- Target file sizes under 200KB for photos
- Use WebP format when possible for better performance
